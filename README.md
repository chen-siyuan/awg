# AMC Worksheet Generator

AWG generates worksheets consisting of problems from [past AMC exams](https://artofproblemsolving.com/wiki/index.php/AMC_Problems_and_Solutions) for **personal practice** purposes.

## Supported Exams

- AMC 8
- AMC 10 (--2001)
- AMC 10A
- AMC 10B
- AMC 12 (--2001)
- AMC 12A
- AMC 12B
- AIME (--1999)
- AIME I
- AIME II
